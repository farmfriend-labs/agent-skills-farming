# Season Extension Planner

Extend growing seasons with techniques and planning.

## Purpose

Growing seasons are short: last frost date limits start, first frost date limits end, shoulder seasons valuable, microclimate variations matter. Season extension extends production by 1-3 months. This skill provides season extension planning.
