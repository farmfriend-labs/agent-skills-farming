# Mixed Fleet Coordinator

Coordinate mixed-brand smaller equipment for efficient operations.

## Purpose

Small operations often have mixed equipment: used John Deere tractor, older Case IH planter, AGCO sprayer, various implements. These systems don't communicate, creating inefficiencies and double work. This skill provides basic coordination tools for mixed-brand smaller fleets.
