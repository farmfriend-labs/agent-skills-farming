# Input Cost Opportunist

Find opportunities for input cost savings through timing and market analysis.

## Purpose

Input costs fluctuate wildly: fertilizer prices vary 30-50% seasonally, seed prices vary 15-25%, chemical prices vary 20-40%, fuel prices vary 30-40%. Buying at the right time saves thousands. This skill monitors input prices and identifies optimal buying windows.
